# 🎵 Decentralized Music Platform (Stellar Soroban)

## 📌 Project Description

The Decentralized Music Platform is a blockchain-powered application built using Stellar Soroban. It enables artists to upload, manage, and share their music without relying on centralized intermediaries like traditional streaming platforms.

This platform ensures transparency, ownership, and fair distribution by leveraging smart contracts, allowing artists to directly connect with their audience.

---

## 🚀 What It Does

This smart contract acts as the backend for a decentralized music ecosystem where:

- Artists can upload music metadata securely on-chain
- Each song is associated with the artist’s wallet address
- Music data (metadata) is stored on-chain while files can be stored off-chain (e.g., IPFS)
- Users can retrieve songs using unique IDs
- The system keeps track of total uploaded songs

---

## ✨ Features

- 🎤 Artist authentication via wallet signature
- 🎵 Upload and store music metadata (title, artist name, URL)
- 🔐 Ownership verification using blockchain addresses
- 🔍 Retrieve songs by unique ID
- 📊 Track total number of uploaded songs
- ⚡ Built on Stellar Soroban for fast and low-cost transactions

---

## 🛠 Tech Stack

- **Blockchain:** Stellar Soroban
- **Language:** Rust
- **Smart Contract SDK:** soroban-sdk
- **Storage:**
  - On-chain: Music metadata
  - Off-chain: IPFS / external storage for audio files
